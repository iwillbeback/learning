python 中一切皆为对象，因此函数参数都是引用传递。只是python对象分为:
可变对象与不可变对象


1> python 中的可变对象: list, dict


2> 不可变对象: int, float, str,tuple
     

3> python 函数缺省参数, 必须为不可变对象
	  python 函数的缺省参数必须为不可变对象，否则会引发如下问题:
     def plus(lst=[]):
     	lst.append(len(lst))
     	
     有如下调用:
     plus()
     第一次输出:  [0]
     
     plus()
     第二次会输出: [0,1]
     
     这是因为如果第一次缺省参数变了， 第二次调用会更改了缺省参数
