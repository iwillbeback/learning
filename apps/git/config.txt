三. GIT配置多个远程主机:

	1>  如果之前配置过global ， 首先取消全局配置
		git config --global --unset user.name
		git config --global --unset user.email
		
	2>  生成二个密钥对, 在ssh 目录下( windows 缺省为: c:/users/xu.wang/.ssh/ )
		ssh-keygen -C "coladou@qq.com" -t -rsa
		

	3> 为每个git 项目分别配置， 也就是在每个的根目录下分别运行 git config 
	   git config --local user.name "xu.wang"
	   git config --local user.email "xu.wang@autonavi.com"
	   
	   git config --local user.name "coladou"
	   git config --local user.email "coladou@qq.com"
	   
	4> 在目录下c:/users/xu.wang/.ssh/, 新建config 文件:
		Host git.debug.amap.com
		HostName git.debug.amap.com
		User xu.wang
		PreferredAuthentications publickey
		IdentityFile ~/.ssh/id_rsa
	 
		Host github.com
		HostName github.com
		User coladou
		PreferredAuthentications publickey
		IdentityFile ~/.ssh/id_rsa_github
