GIT 的几个基本概念
1.  版本库 
	  一般指项目根目录下的.git 目录
	  用于保存所有元配置和历史信息
	  相关命令：
	   git log: 查看版本库的更新内容

3. 索引区
     是一个在工作目录 和 版本库之间的暂存区。
     相关命令
      1> git status: 查看工作目录，和索引区的更新内容
      2> git add   : 将工作目录更新，添加到索引区
      3> git commit: 将索引区更新，添加到版本库

2. 工作区目录
	 用于存储你当前分支内容
	 当在不同分支之间切换时，工作目录内容会被不断删除和替换
	 而所有分支内容及历史信息都保存在GIT 目录下
