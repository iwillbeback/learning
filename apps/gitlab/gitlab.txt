Gitlab 本地环境搭建步骤:

1.  Windows 下安装客户端msysgit
      从http://msysgit.github.io/下载，然后按默认选项安装即可
      安装完成后，在开始菜单里找到“Git”->“Git Bash”，就说明Git安装成功！

2.  配置本地用户名和邮箱
     在git bash下进入d:/working/learning目录
     1> git config --local user.name "coladou"
	 2> git config --local user.email "coladou@qq.com"
     配置完成后查看当前配置信息:
  	git config --local -l
  
	 
3. 在github 远端配置ssh key
     1>  生成密钥对, 打开git bash
		    ssh-keygen -t rsa -C "coladou@qq.com"
		    
	 2> 将生成的公钥文件内容添加至gitlab:
	    选择"setting"--> "ssh"
	 
4. 得到一个GIT库,  有两种情况:
	 1> 从远端克隆已有github 库
	 2> 从本地创建一个git 库， 将推送至github
	 以下5~7为前者 

5.  创建&初始化本地仓库
     选定或创建一个目录，如d:/working/learning, 在gib bash环境下:
      git init
	
6.  在github创建远端版本库, 并在本地分支做关联
      git remote add origin git@github.com:coladou/learning.git
      
7.  将本地分支推送到github
	  git push -u origin master --force
